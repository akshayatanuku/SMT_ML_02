Task 02: Customer Segmentation using K-MeansClusteringThis project applies Unsupervised Machine Learning to group retail customers based on their purchasing habits and demographic data.
Key Highlights
Algorithm: Implemented K-Means Clustering to identify distinct customer segments within the mall_customers.csv dataset.
Optimal Clusters: Used the Elbow Method to determine the ideal number of clusters ($K$) by calculating the Within-Cluster Sum of Squares (WCSS).
Data Analysis: Analyzed features like Annual Income and Spending Score to differentiate between "High Spenders," "Budget Conscious," and "Target Customers.
"Visualization: Created 2D and 3D scatter plots using Seaborn and Matplotlib to visually represent the discovered clusters.
Goal: To help businesses develop targeted marketing strategies by understanding customer diversity and behavior.
